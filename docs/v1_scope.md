## Goal
Build a production-grade personal finance system that:
- Tracks spending
- Summarizes financial behavior
- Explains decisions in plain English

This project prioritizes **engineering quality** over feature count.

## In Scope (v1)
- Manual transaction upload (CSV)
- Transaction categorization
- Monthly spending summaries
- AI-generated spending explanations
- Backend API
- Minimal frontend UI

## Out of Scope (v1)
- Bank integrations (Plaid)
- Real-time syncing
- Budget automation
- Investment tracking
- User authentication

## Success Criteria
- Deployed backend and frontend
- Clear README explaining architecture
- Passing automated tests
